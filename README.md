# Flask-app-ansible 🚀

This project is a simple **Flask web application** containerized using **Docker**, and automated for deployment using **Ansible**. It demonstrates how to serve a basic web app efficiently in a DevOps environment.

---

## 🧰 Tech Stack

- Python 3
- Flask
- Docker
- Ansible
- Ubuntu Server (EC2 or any Linux-based VM)
